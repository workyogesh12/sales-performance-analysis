# Sales Performance Analysis Dashboard

## Project Objective

This project analyzes sales data of a small online store to understand business performance using key metrics and visualizations.

## What This Project Analyzes

- Total sales
- Total profit
- Total orders
- Best performing region
- Worst performing region
- Sales by category
- Profit by region
- Monthly sales trend
- Top 10 products by sales

## Tools Used

- HTML
- CSS
- JavaScript
- Chart.js
- GitHub Pages

## Dataset

The dataset contains sample sales records with the following columns:

- Order ID
- Order Date
- Region
- Category
- Product
- Sales
- Profit
- Quantity

## Key Insights

- Technology category generated strong sales.
- The best performing region is identified using total sales.
- Monthly sales trend helps understand growth patterns.
- Top products help the business focus on high-revenue items.
- Profit by region helps identify strong and weak locations.
